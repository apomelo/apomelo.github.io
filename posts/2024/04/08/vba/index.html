VBA(Visual Basic for Applications) 笔记

发表于 2024-04-08 分类于 language 阅读次数：

简介

VBA，全名 Visual Basic for Applications，是 Microsoft 为其 Office 系列产品如 Excel、Word、Access 等提供的一种宏语言。它是 Visual Basic 的一个子集，主要用于自动化 Office 应用程序的任务，包括创建用户定义的函数、自动执行复杂任务、创建和管理 Windows 应用程序等。

VBA 是一种面向对象的编程语言，可以创建复杂的用户界面和其他高级功能。它具有强大的编程能力，可以访问 Windows API 和其他低级功能，同时还可以使用 SQL 连接数据库。

VBA 的主要优点是易于学习和使用，尤其是对于那些已经熟悉 Microsoft Office 的用户。此外，由于它是 Office 的内置组件，因此无需另外安装就可以使用。然而，它的缺点是运行速度相对较慢，且只能在 Windows 环境下运行。

显示 “开发工具” 选项卡

需要编写或录制 Excel 宏，需要打开 “开发工具” 选项卡

默认情况下不显示“ 开发工具 ”选项卡，但你可以将其添加到功能区。

  1. 在 “文件” 选项卡上，转到 “选项” > “自定义功能区”。
  2. 在 “自定义功能区” 和 “主选项卡” 下，选中 “开发工具” 复选框。

显示选项卡后，除非清除“检查”框或必须重新安装 Microsoft 365 程序，否则**“开发工具**”选项卡将保持可见。

“开发工具” 选项卡是你想要执行或使用以下项时转到的位置：

  • 编写宏。
  • 运行以前录制的宏。
  • 使用 XML 命令。
  • 使用 ActiveX 控件。
  • 创建用于 Microsoft 365 程序的应用程序。
  • 在 Microsoft Excel 中使用表单控件。
  • 在 Microsoft Visio 中使用形状表。
  • 在 Microsoft Visio 中创建新的形状和模具。

VBA 语言的基础认识

书写规范

  1. VBA 不区分标识符的字母大小写，一律认为是小写字母；
  2. 一行可以书写多条语句，各语句之间以冒号 : 分开；
  3. 一条语句可以多行书写，以空格加下划线 _ 来标识下行为续行；
  4. 标识符最好能简洁明了，不造成歧义。

标识符

定义

标识符是一种标识变量、常量、过程、函数、类等语言构成单位的符号，利用它可以完成对变量、常量、过程、函数、类等引用。

命名规则

（1）字母打头，由字母、数字和下划线组成，如A987b_23Abc

（2）字符长度小于 40，（Excel2002 以上中文版本等，可以用汉字且长度可达 254 个字符）

（3）不能与 VB 保留字重名，如 public，private，dim，goto，next，with，integer，single 等

运算符

定义：运算符是代表 VB 某种运算功能的符号。

（1）赋值运算符 =

（2）数学运算符 &（强制字符连接）、+ （运算符两边都是字符时连接字符）、+（运算符两边都是数字时为加法运算符）、-（减）、Mod（取余）、\（整除）、*（乘）、/（除）、-（负号）、^（指数）

（3）逻辑运算符 Not（非）、And（与）、Or（或）、Xor（异或）、Eqv（相等）、Imp（隐含）

（4）关系运算符 = （相同）、<>（不等）、>（大于）、<（小于）、>=（不小于）、<=（不大于）、Like（判断给定的字符串是否与指定的模式相匹配）、Is（判断该数据是数字还是是字符）

（5）位运算符  Not（逻辑非）、And（逻辑与）、Or（逻辑或）、Xor（逻辑异或）、Eqv（逻辑等）、Imp（蕴含）

变量与常量

（1）VBA 允许使用未定义的变量，默认是变体变量

（2）在模块通用说明部分，加入 Option Explicit 语句可以强迫用户进行变量定义

（3）变量定义语句及变量作用域

一般变量作用域的原则是，在哪部分定义就在哪部分起作用，模块中定义则在该模块起作用。

（4）常量为变量的一种特例，用 Const 定义，且定义时赋值，程序中不能改变值，作用域也如同变量作用域。如下定义：Const Pi=3.1415926 as single

常用变量定义示例

在 VBA（Visual Basic for Applications）中，变量是用于存储数据的标识符。在使用变量之前，需要使用Dim语句声明它们。以下是 VBA 中常用的变量类型和示例：

  1. String （字符串类型）：用于存储文本数据。
  2. Boolean （布尔类型）：用于存储逻辑值，即 True 或 False。
  3. Byte （布尔类型）：用于存储 Byte 值，范围为-128 到 127。
  4. Integer （整数类型）：用于存储整数值，范围为-32,768 到 32,767。
  5. Long （长整型）：用于存储较大范围的整数值，范围为-2,147,483,648 到 2,147,483,647。
  6. Single （单精度浮点类型）：用于存储单精度浮点数，即具有小数点的数字。
  7. Double （双精度浮点类型）：用于存储双精度浮点数，比单精度更精确。
  8. Date （日期类型）：用于存储日期和时间数据。
  9. Object （对象类型）：用于引用 Excel 工作表、范围或其他对象。

文件操作

文件

  1. Dir[(pathname[, attributes])] :

    • path 是要搜索的目录路径。如果省略 path 参数，则 Dir 返回当前目录中符合条件的第一个文件或目录。如果没有找到  pathname，则会返回零长度字符串 (“”);
    • attributes 是一个可选参数，用于指定要匹配的文件或目录的属性。它可以是以下常量之一：
      • vbNormal：普通文件。
      • vbReadOnly：只读文件。
      • vbHidden：隐藏文件。
      • vbSystem：系统文件。
      • vbDirectory：目录或文件夹。
      • vbArchive：归档文件。

    Dir 函数返回符合条件的第一个文件或目录的名称，如果没有找到匹配的文件或目录，则返回空字符串。每次调用 Dir 函数后，它都会记住上次搜索的位置，以便下一次调用时继续搜索。如果要重新开始搜索，可以将 Dir 函数的返回值设置为空字符串。

删除

  1. Kill pathname : 从磁盘中删除文件, pathname 参数是用来指定一个文件名
  2. RmDir pathname : 从磁盘中删除目录，pathname 参数是用来指定一个文件夹

打开

  1. Open pathname For mode [Access access] [lock] As [#]filenumber [Len=reclength] : 能够对文件输入/输出 (I/O)。

    • pathname 必要。字符串表达式，指定文件名，该文件名可能还包括目录、文件夹及驱动器。
      mode 必要。关键字，指定文件方式，有 Append、Binary、Input、Output、或 Random 方式。如果未指定方式，则以 Random 访问方式打开文件。
    • access 可选。关键字，说明打开的文件可以进行的操作，有 Read、Write、或 ReadWrite 操作。
    • lock 可选。关键字，说明限定于其它进程打开的文件的操作，有 Shared、LockRead、LockWrite、和 Lock Read Write 操作。
    • filenumber 必要。一个有效的文件号，范围在 1 到 511 之间。使用 FreeFile 函数可得到下一个可用的文件号。
    • reclength 可选。小于或等于 32，767（字节）的一个数。对于用随机访问方式打开的文件，该值就是记录长度。对于顺序文件，该值就是缓冲字符数。

    说明： 对文件做任何 I/O 操作之前都必须先打开文件。Open   语句分配一个缓冲区供文件进行 I/O 之用，并决定缓冲区所使用的访问方式。如果 pathname 指定的文件不存在，那么，在用 Append、Binary、Output、或 Random 方式打开文件时，可以建立这一文件。如果文件已由其它进程打开，而且不允许指定的访问类型，则 Open 操作失败，而且会有错误发生。如果 mode 是 Binary 方式，则 Len 子句会被忽略掉。

    重要： 在 Binary、Input 和 Random 方式下可以用不同的文件号打开同一文件，而不必先将该文件关闭。在 Append 和 Output 方式下，如果要用不同的文件号打开同一文件，则必须在打开文件之前先关闭该文件。

读入

  1. Input #filenumber, varlist : 从已打开的顺序文件中读出数据并将数据指定给变量
  2. Get [#]filenumber, [recnumber], varname : 将一个已打开的磁盘文件读入一个变量之中。

写入

  1. Write #filenumber, [outputlist] : 将数据写入顺序文件
  2. Print #filenumber, [outputlist] : 将格式化显示的数据写入顺序文件中
  3. Put [#]filenumber, [recnumber], varname : 将一个变量的数据写入磁盘文件中。

关闭

Close [filenumberlist] 关闭 Open 语句所打开的输入/输出 (I/O) 文件

注意:

如果今后想用 Input # 语句读出文件的数据，就要用 Write # 语句而不用 Print # 语句将数据写入文件。因为在使用 Write # 时，将数据域分界就可确保每个数据域的完整性，因此可用 Input # 再将数据读出来。使用 Write # 还能确保任何地区的数据都被正确读出。Write 与 Print # 语句不同，当要将数据写入文件时，Write # 语句会在项目和用来标记字符串的引号之间插入逗号。Write # 语句在将 outputlist 中的最后一个字符写入文件后会插入一个新行字符，即回车换行符，(Chr(13) + Chr(10))。

其他文件函数

  1. LOF(filenumber) : 返回一个 Long，表示用 Open 语句打开的文件的大小，该大小以字节为单位。
  2. EOF(filenumber) : 返回一个 Integer，它包含 Boolean 值 True，表明已经到达为 Random 或顺序 Input 打开的文件的结尾。
  3. Loc(filenumber) : 返回一个 Long，在已打开的文件中指定当前读 / 写位置
  4. Seek(filenumber) : 返回一个 Long，在 Open 语句打开的文件中指定当前的读 / 写位

直接用 Shell 执行

Shell 函数是 VBA 提供的一个内置函数，用于在操作系统中执行外部命令或程序。它的使用方式非常简单，只需提供要执行的命令字符串即可。

优点是简单易用，适用于执行简单的命令。但是它对于捕获命令执行的输出并不方便，因为它无法直接返回命令的输出结果。

由于 Shell 函数本身并不返回命令的输出结果。所以要获取命令执行的结果，需要通过其他方法，例如使用文件来保存输出，然后再读取文件内容。

使用 CreateObject(“WScript.Shell”)

WshShell 对象属于 Windows Script Host（WSH）的一部分，通过该对象可以执行命令并捕获命令执行的输出。需要先引入 “Windows Script Host Object Model” 库，然后通过 CreateObject 函数创建 WshShell 对象

如果只是简单地执行外部命令，并不需要捕获输出，直接使用 Shell 函数即可；如果需要捕获命令的输出并进行后续处理，可以考虑使用 WshShell 对象。

windows cmd 命令如何同时运行两个命令

在 Windows 的命令提示符（cmd）中，你可以使用以下方法同时运行两个命令：

使用 & 符号： 使用 & 符号可以将两个命令连接起来，使它们同时运行。

command1 & command2

例如：

dir & echo "Both commands executed"

使用 && 符号： 使用 && 符号也可以将两个命令连接起来，但是第一个命令成功执行后才会运行第二个命令。

command1 && command2

例如：

echo "Command 1 succeeded" && echo "Command 2 executed"

使用 || 符号： 使用 || 符号可以在第一个命令失败时运行第二个命令。

command1 || command2

例如：

del non_existent_file.txt || echo "File not found"

这些方法允许你在 Windows 命令提示符中同时运行多个命令，可以根据需要选择适合的连接符号来控制命令的执行顺序。

在其他程序中调用

需要在命令前加上 cmd 命令

例如：

cmd /c command1 && command2
