配置 git

配置文件位置

Git相关的配置文件有三个：

  1. 系统级
    用户目录中的 .gitconfig 文件
  2. 系统用户级
    用户目录中的 .gitconfig 文件
  3. 仓库级
    git项目目录中的 .git/config 文件

对于同一配置项，三个配置文件的优先级是 系统级 < 系统用户级 < 仓库级
