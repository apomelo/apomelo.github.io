简介

Python 是一种编程语言，广泛用于 Web 应用程序、软件开发、数据科学和机器学习(ML)。 开发人员使用 Python 是因为其高效和易于学习，并且可以在许多不同的平台上运行。 Python 软件可供免费下载，可与所有类型的系统完美集成，并且还可以提高开发速度。

在编程过程中，用 python 进行一些工具、脚手架等等一些小东西是非常方便的。

便携版 python + pip 使用指南

以 windows 系统示例，其他系统大同小异

下载 python

python 下载链接： https://www.python.org/downloads/windows/

Download Windows embeddable package (64-bit)

安装 pip

下载文件 get-pip.py 到便携版 python 的目录下，然后在当前目录下打开 cmd，写入命令 ./python get-pip.py 并敲击回车运行命令

get-pip.py 下载地址：

https://bootstrap.pypa.io/get-pip.py

安装完成之后目录下看会发现多了两个文件夹，Lib 和 Script

点进 Script 文件夹去看发现有各种各样的 pip——pip.exe/pip3.exe/pip3.11.exe。

注意！！！ 这时候这些命令还不能使用，还需要进行下一步修改：

修改 python<版本号>._pth (这里下载的版本是 3.11.3，所以文件名称是 python311._pth) 文件中的 import site，去掉前面的注释，也就是 #。

这里的 311 可以参考上面 Script 文件夹里面的 pip3.11.exe
